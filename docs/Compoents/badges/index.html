Badges

Badges helps to notify user spicifically on component.

Examples#

Inline Badge#

Notifications

</button>

Overlay Badge#

<button class="sw-btn sw-btn-surface m-4">
Mail
<div class="sw-badge sw-badge-pill sw-badge-overlay rounded" >10</div>
</button>
